Anmeldung

Die Beitragsrunde benötigt keine Passwörter. Einfach die E-Mail-Adresse angeben und den Anmeldelink im Postfach öffnen.

Beitragsrunde anlegen

Eine Beitragsrunde hat einen Abstimmungszeitraum und beliebig viele Produkte.

Produkte

  • Sobald der Abstimmungszeitraum beginnt, können die Mitglieder ihre Gebote abgeben
  • Die Mitglieder können pro Produkt so viele Gebote abgeben, wie Anzahl an Runden konfiguriert werden. Es können im Nachhinein noch weitere Runden hinzugefügt werden
  • Der Richtwert ist der benötigte Betrag für das Gartenjahr
  • Alle zuvor angelegten Benutzer werden automatisch eingeladen
  • Es kann immer nur eine Beitragsrunde durchgeführt werden. Eine Beitragsrunde kann aber beliebig viele Produkte enthalten

Benutzer anlegen

Um einen Benutzer anlegen zu können, ist nur ein Name und eine E-Mail-Adresse notwendig. Sonst nichts.

Benutzerrollen

  • Admin: Kann Beitragsrunde und Benutzer verwalten
  • Benutzer: Kann an einer Beitragsrunde teilnehmen

Benutzer anlegen - Massenimport

Benutzer können auch via xlsx oder csv importiert werden.

Einfach diese Vorlage befüllen und importieren: Benutzervorlage

Bei doppelten E-Mail-Adressen wird nur ein Benutzer angelegt.

Gebote abgeben

Der Benutzer kann die Gebote abgeben, sobald der Abstimmungszeitraum begonnen hat. Der Benutzer sieht die Produkte und Anzahl an Runden, die zuvor konfiguriert wurden.

Solange noch keine Runde ermittelt wurde, kann der Benutzer seine Gebote neu abgeben.

Gebote für Benutzer abgeben

Admins können im Namen der Benutzer Gebote abgeben.

Runde ermitteln

Sobald alle Gebote abgegeben wurden, ermittelt das System die Runde mit genügend Umsatz.

Gebote exportieren

Um die Gebote weiter verarbeiten zu können, können sie exportiert werden. Folgende Formate sind dabei möglich:

  • xlsx
  • csv
  • pdf

Benutzer aus Beitragsrunde entfernen

Um irrtümlich hinzugefügte Benutzer zu entfernen, kann der Admin die Benutzer aus der Beitragsrunde entfernen.
