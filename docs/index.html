Beitragsrunde - Erste Schritte

Für die Beitragsrunde brauchst du kein Passwort. Gib einfach deine E-Mail-Adresse ein und öffne den Anmeldelink, der in dein E-Mail-Postfach geschickt wird.

Um einen neuen Benutzer anzulegen, brauchst du nur den Namen und die E-Mail-Adresse der Person - mehr nicht.

Benutzerrollen

  • Admin: Kann Beitragsrunden und Benutzer verwalten.
  • Benutzer: Kann an einer Beitragsrunde teilnehmen.

Benutzer können auch über .xlsx- oder .csv-Dateien importiert werden.

Benutzervorlage

Hinweis: Bei doppelten E-Mail-Adressen wird nur ein Benutzer angelegt.

Eine Beitragsrunde hat einen Abstimmungszeitraum und beliebig viele Produkte.

Produkte

  • Sobald der Abstimmungszeitraum beginnt, können die Mitglieder ihre Gebote abgeben.
  • Die Mitglieder können pro Produkt so viele Gebote abgeben, wie Runden konfiguriert wurden. Weitere Runden können nachträglich hinzugefügt werden.
  • Der Richtwert ist der benötigte Betrag für das Gartenjahr.
  • Alle zuvor angelegten Benutzer werden automatisch eingeladen.
  • Es kann immer nur eine Beitragsrunde durchgeführt werden, diese kann jedoch mehrere Produkte enthalten.

Du kannst deine Gebote abgeben, sobald der Abstimmungszeitraum begonnen hat. Du siehst die Produkte und die Anzahl der konfigurierten Runden.

Solange noch keine Runde ermittelt wurde, kannst du deine Gebote erneut abgeben.

Admins können bei Bedarf Gebote im Namen der Benutzer abgeben.

Falls Benutzer irrtümlich hinzugefügt wurden, können Admins diese aus der Beitragsrunde entfernen.

Sobald alle Gebote abgegeben wurden, ermittelt das System die Runde mit genügend Umsatz.

Um die Gebote weiterverarbeiten zu können, kannst du sie in verschiedenen Formaten exportieren:

  • .xlsx
  • .csv
  • .pdf
